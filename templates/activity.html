Requirements

  • JavaScript enabled browser
  • Ability to print to a PDF file
    OSX can do this natively. Windows users will need to use a browser with an appropriate plugin.
    Learn more about saving/printing to PDF

Saving

To save your work, print your page to a PDF.

Using Mac OSX

  1. Start the print process
  2. In the "PDF" menu, select "Save as PDF..."

Using Firefox

  1. Install Print pages to PDF
  2. Right-click on the page and select "Print page to PDF"
